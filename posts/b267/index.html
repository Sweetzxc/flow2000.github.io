如何申请一个永久免费的Mongodb数据库-详细版

发表于2022-09-25|更新于2022-09-27|Mongodb
|字数总计:658|阅读时长:2分钟|阅读量:|评论数:

简介

MongoDB是一个文档数据库，它类似于JSON对象，我们可以使用MongoDB存储我们的博客数据，比如用户评论，友链文章信息等等，但由于自建服务略有些麻烦，今天教大家如何申请一个永久免费的Mongodb数据库供我们使用。twikoo以及友链朋友圈都可以基于以下教程使用MongoDB

注册账号

MongoDB Cloud Register

填写邮箱等信息，验证邮箱后选择FREE免费套餐

创建数据库用户

设置自己想要的数据库用户和密码

选择地区

选择提供商选择AWS，地区选择Hong Kong (ap-east-1)，点击Create Cluster

创建完成后进入到控制台界面

连接字符串

点击Connect，MongoDB提供了四种连接方式，我们只用两种就行（Connect using VS Code、Connect your application）

VSCode连接

使用VSCode连接可以快速查看和操作可视化数据

在上一步选择Connect using VS Code，复制字符串备用

安装VSCode插件MongoDB for vs Code

点击左边的小叶子，点击加号，点击页面的Connect，出现输入框后粘贴连接字符串，记得将<password>替换成原先创建的数据库密码（有<user>的话替换成创建的数据库账号），回车等待连接完成

成功截图：

应用连接

上述步骤只是我们便于查看和操作可视化数据，我们还需要一个连接字符串用于应用开发，此时我们选择Connect your application，选择驱动语言为python，twikoo以及友链朋友圈都是基于python开发的。复制字符串，并替换<password>（有<user>的话替换成创建的数据库账号），这个字符串就是进行开发时需要用到的字符串。

Studio 3T连接

如果VSCode不满足可视化数据的需求，可以使用Studio3T连接，连接字符串使用应用连接的字符串即可。

MongoDB Compass连接

MongoDB Compass是Mongodb官方的可视化管理工具，不仅拥有数据的基本操作，而且支持搜索、表格与 Json 格式查看，一键导入导出 Json 或 csv等等

下载方式可在选择连接方式的第三钟方式获得：Connect using MongoDB Compass，选择windows版本下载即可，连接字符串同样选择应用连接

再次感谢博主张时贰的指点，这个官方自带的连接工具的确方便了数据操作，也减少了卡顿的产生（Studio 3T有时候很卡顿）
